# Evaluation of AI Chatbot using RAGAS



## Tech Stack

Python 3, RAGAS, Langchain



## Steps to Run
1. Make sure you have Python 3 or higher installed on your system.

2. Add your USERNAME, PASSWORD, and your CHATBOT_URL and LLM configurations to the .env file.

3. Add your files to the data_files_for_KB directory.

4. Ensure you are adding the same files that were used to create the knowledge base for your chatbot.

5. Run the command python runner.py or python3 runner.py in your project terminal.

6. After code execution, you can view the result metrics in the ragas_report folder in JSON format.
